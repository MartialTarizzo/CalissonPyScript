Présentation

Cette page propose des grilles du jeu du Calisson, inventé par Olivier LONGUET.

Pour connaître la règle du jeu, s'informer sur les techniques de résolution et se confronter aux grilles proposées par l'auteur, aller sur le site du jeu.

Les grilles proposées ici sont calculées à l'aide d'un ensemble de fonctions programmées en Python, dont le code de référence est disponible à l'adresse suivante : https://github.com/MartialTarizzo/calisson

Pour utiliser le code Python, la plate-forme PyScript est utilisée : elle permet de compiler le code Python en un ensemble d'instructions exécutables par le navigateur.

Normalement, toutes les grilles ont une solution unique. Si jamais ce n'était pas le cas, c'est qu'il y a un erreur dans le code Python utilisé lors de la génération de ces grilles. Ne pas hésiter alors à m'envoyer un mail à l'adresse suivante :
martial.tarizzo (at) gmail.com

Mode d'emploi

  • sélectionner la taille de grille ainsi que sa "facilité" : plus la grille est facile, plus il y a d'arêtes.
  • Cliquer sur le bouton "Créer une nouvelle grille" afin de jouer sur une grille ayant les caractéristiques sélectionnées. Soyez patient pour les grilles de grande taille : le calcul peut prendre plusieurs secondes ...
  • Il est possible de 'tricher' (à n'utiliser qu'en dernière extrémité, quand plus rien ne va...) :
    cliquer dans la zone de jeu avec la touche "Alt" du clavier enfoncée va afficher la solution !

Pour afficher de nouveau ces informations, il suffit de cliquer sur le bouton "Mode d'emploi".
